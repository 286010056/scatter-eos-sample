• 什么是 DAPP 挖矿

    通过不断自动发送玩游戏交易获得游戏 Token 奖励。当玩的次数足够大时，概率稳定，通过获得的 Token 价值超过游戏支付的 EOS 开支从而获得收益。由于 Token 价格随市场变化，所以挖矿也有风险，请谨慎参与。
