### 测试方法

在钱包的 DAPP 中找到【麦子 DAPP 浏览器】应用，粘贴地址即可进行测试。

### 如何切换 EOS 测试网络

1 在代码中更换 network 参数为测试网络

2 在麦子钱包 - 我 - 设置 - 节点设置 - EOS - 添加自定义节点 - 选中添加的测试节点
